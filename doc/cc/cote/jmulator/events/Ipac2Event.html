Packagecc.cote.jmulator.events
Classpublic class Ipac2Event
InheritanceIpac2Event EmulatorEvent flash.events.Event

Since : version 2.0b

An Ipac2Event object is dispatched whenever the status of an input on an Ipac2 board changes. The name of the dispatched event is exactly the name of the input (as labeled on the board itself).
